<span data-ttu-id="c438b-114">*サイクロマティック複雑度* は、メソッドを使用して線形的に独立したパスの数を測定します。これは、条件分岐の数と複雑さによって決まります。</span><span class="sxs-lookup"><span data-stu-id="c438b-114">*Cyclomatic complexity* measures the number of linearly independent paths through the method, which is determined by the number and complexity of conditional branches.</span></span> <span data-ttu-id="c438b-115">サイクロマティック複雑度が低いのは、一般に、理解、テスト、および保守が容易なメソッドであることを示しています。</span><span class="sxs-lookup"><span data-stu-id="c438b-115">A low cyclomatic complexity generally indicates a method that is easy to understand, test, and maintain.</span></span> <span data-ttu-id="c438b-116">サイクロマティック複雑度は、メソッドの制御フローグラフから計算され、次のように指定されます。</span><span class="sxs-lookup"><span data-stu-id="c438b-116">The cyclomatic complexity is calculated from a control flow graph of the method and is given as follows:</span></span>

<span data-ttu-id="c438b-117">サイクロマティック複雑度 = エッジの数-ノードの数 + 1</span><span class="sxs-lookup"><span data-stu-id="c438b-117">cyclomatic complexity = the number of edges - the number of nodes + 1</span></span>

<span data-ttu-id="c438b-118">*ノード* はロジックの分岐点を表し、*エッジ* はノード間の線を表します。</span><span class="sxs-lookup"><span data-stu-id="c438b-118">A *node* represents a logic branch point and an *edge* represents a line between nodes.</span></span>

<span data-ttu-id="c438b-119">このルールは、サイクロマティック複雑度が25を超える場合に違反を報告します。</span><span class="sxs-lookup"><span data-stu-id="c438b-119">The rule reports a violation when the cyclomatic complexity is more than 25.</span></span>

<span data-ttu-id="c438b-120">コードメトリックスの詳細については、「 [マネージコードの複雑さの測定](/visualstudio/code-quality/code-metrics-values)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="c438b-120">You can learn more about code metrics at [Measure complexity of managed code](/visualstudio/code-quality/code-metrics-values).</span></span>

        Console.WriteLine("Hello World!")
    End If

### <a name="cyclomatic-complexity-of-3"></a><span data-ttu-id="c438b-135">サイクロマティック複雑度3</span><span class="sxs-lookup"><span data-stu-id="c438b-135">Cyclomatic complexity of 3</span></span>

    End If

            Console.WriteLine("Today is Tuesday!");
            break;
        case DayOfWeek.Wednesday:
            Console.WriteLine("Today is Wednesday!");
            break;
        case DayOfWeek.Thursday:
            Console.WriteLine("Today is Thursday!");

            break;
        case DayOfWeek.Saturday:

Public Sub Method(ByVal day As DayOfWeek)

        Case DayOfWeek.Wednesday

            Console.WriteLine("Today is Thursday!")
        Case DayOfWeek.Friday
